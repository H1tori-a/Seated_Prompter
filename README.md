### Seated_Prompter

# 所需硬件
1.	3D打印外壳 
2.	Arduino_UNO控制器                1个
3.	超声波                            1个
4.	LED灯                             2个
5.	数码管                            1个
6.	语音模块                          1个
7.	9V电池                           1个
8.	导线
